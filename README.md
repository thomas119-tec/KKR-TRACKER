# Personal Finance Tracker

A simple, modern web application for tracking personal finances, including daily spends, income, categorizing expenses, and tracking borrowed/lent money.

## Features

- Add daily transactions (spends and receives) with categories
- Track money borrowed from you (lent) and money you borrowed
- Simple, modern UI with a clean design
- Data stored locally in your browser

## How to Use

1. Open `index.html` in your web browser.
2. Add transactions using the "Transactions" form.
3. Add borrow/lend records using the "Borrows" form.
4. View your summary at the bottom.

## Categories

Available expense categories:
- Food
- Transport
- Entertainment
- Utilities
- Other

## Data Storage

All data is stored locally in your browser's localStorage. It will persist between sessions but will be cleared if you clear your browser data.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## Browser Compatibility

Works in all modern browsers that support localStorage.

## Troubleshooting

- If data doesn't save, check if your browser has localStorage enabled.
- For any issues, try refreshing the page or clearing browser cache.
